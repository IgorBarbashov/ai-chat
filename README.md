# AI Chat (GigaChat)

ChatGPT-style интерфейс для работы с [GigaChat API](https://developers.sber.ru/portal/products/gigachat).

## Быстрый старт

### 1. Установка зависимостей

```bash
npm install
```

### 2. Настройка токена GigaChat

Создайте файл `.env.local` в корне проекта:

```
VITE_GIGACHAT_TOKEN=ваш_bearer_токен
```

Токен получают в личном кабинете GigaChat. Время жизни токена — около 30 минут, после истечения нужно обновить значение в `.env.local` и перезапустить dev-сервер.

### 3. Запуск

```bash
npm run dev
```

Приложение откроется на `http://localhost:5173`.

## Команды

| Команда | Описание |
|---|---|
| `npm run dev` | Dev-сервер с HMR |
| `npm run build` | Сборка для продакшена |

## Реализованный функционал

### Интерфейс чата

- Отображение сообщений пользователя и ассистента в хронологическом порядке с визуальным разделением
- Поддержка Markdown-форматирования в ответах (заголовки, списки, блоки кода, ссылки)
- Подсветка синтаксиса в блоках кода (highlight.js)
- Индикатор печати (typing indicator) во время ожидания ответа
- Автоматическая прокрутка к последнему сообщению
- Копирование ответа ассистента в буфер обмена
- Кнопка «Стоп» для прерывания генерации

### Управление чатами

- Боковая панель со списком чатов и поиском по названию
- Создание нового чата
- Автогенерация названия чата на основе первого сообщения
- Переключение между чатами без потери данных
- Редактирование названия чата (inline)
- Удаление чата с подтверждением
- Сохранение истории чатов в `localStorage` — данные восстанавливаются после перезагрузки страницы

### Интеграция с GigaChat API

- Отправка запросов к `POST /api/v1/chat/completions` с корректными заголовками
- Передача полного контекста диалога (массив `messages` с ролями `user` / `assistant`)
- Потоковый режим через SSE (`stream: true`) — ответ отображается токен за токеном
- Обработка ошибок API с отображением уведомления

## Не реализовано

| Функция | Причина |
|---|---|
| Автообновление OAuth-токена | Ручная ротация достаточна для dev-режима |
| Прикрепление изображений (multimodal) | Выходит за рамки текущего объёма |
| Поиск по содержимому чатов | Реализован поиск по названию |

## Технологии

- **React 19** + **TypeScript 5**, сборка на **Vite 8**
- **Mantine 8** — UI-компоненты и темизация
- **react-markdown** + **rehype-highlight** — рендеринг и подсветка кода
- **Fetch API** — SSE-стриминг через `ReadableStream`
