- в импортах используй aliases @components
- при формировании папки для компонента используй следующий соглашения (по набору файлов, наименованию и стилю намиенования файлов, содержанию файлов):
    - сам компонент каходится в файле <ComponentName>.tsx
    - интерфейсы и типы TS для компонента находятся в отдельном файле <componentName>.models.ts
    - стили находятся в отдельном файле <componentName>.module.css
    - кастомные хуки находятся в отдельном файлах <useFunction>.ts (один файл - один хук, если хуков больше одного, они выносятся в подпапку hooks и импортируются из нее через public api - index.ts)
    - все файлы втури папки компонента импортируются через относительные пути
    - компоненты между собой импортируются через public api копонента - index.ts
