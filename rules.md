Проанализируй проект. Нужно добавить Область чата (ChatWindow)
Компонент ChatWindow (центральная зона) должен включать:
- заголовок с названием текущего чата и кнопкой настроек;
- область сообщений MessageList с вертикальной прокруткой;
- минимум 6 моковых сообщений (3 — от пользователя, 3 — от ИИ-ассистента) с визуальным разделением:
- сообщения пока отображаются просто div, потом для них разработаем отдельный компонент
- сообщения пользователя располагаются справа, имеют отличный от основного цвет фона;
- сообщения ассистента располагаются слева, имеют аватар или иконку GigaChat.
- расположение: components\chat\ChatWindow.tsx
- родительский компонент - AppLayout.tsx
- получение данных и вся логика вынесена в кастомные хуки (лучше по возможности разнести в разные - отдельно получение данных - пока замокано и отдается просто массив с имитацией асинхронности с помощью setTimeout, отдельно всякие локальные state и handlers)
- получение данных тригерится на изменение id активного чата (это происходит при нажатии на ChatItem в SideBar), activeChatId сейчас хранится в моковых данных
- сейчас не надо реализовывать возможность изменения этого activeChatId, на надо учесть реакцию на него


-------------

Общие требования ко всему проекту:
- в импортах используй aliases @components
- при формировании папки для компонента используй следующий соглашения (по набору файлов, наименованию и стилю намиенования файлов, содержанию файлов):
    - сам компонент каходится в файле <ComponentName>.tsx
    - интерфейсы и типы TS для компонента находятся в отдельном файле <componentName>.models.ts
    - стили находятся в отдельном файле <componentName>.module.css
    - кастомные хуки находятся в отдельном файлах <useFunction>.ts (один файл - один хук, если хуков больше одного, они выносятся в подпапку hooks и импортируются из нее через public api - index.ts)
    - все файлы втури папки компонента импортируются через относительные пути
    - компоненты между собой импортируются через public api копонента - index.ts
- инструкции по code-style:
    - if всегда пиши в полном варианте (с фигурнвми скобками) при любом количестве строк
